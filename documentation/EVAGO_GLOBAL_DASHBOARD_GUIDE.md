## üöÄ Quick Start Guide

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Generate Global Marketing Data
```bash
cd code
python evago_global_marketing_simulator.py
```

### 3. Prepare Power BI Data
```bash
python powerbi_global_preparation.py
```

### 4. Import to Power BI
1. Open Power BI Desktop
2. Import each `powerbi_chart*.csv` file from `powerbi_dashboards/`
3. Create relationships between datasets
4. Build the 10 charts as described above
